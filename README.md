## LeinaoPAI
### 欢迎大家将镜像的dockerfile分享出来

当有编译镜像需求时，提交pull request到本项目，要求：
1. 统一放置在images目录下
2. 每个镜像一个子目录，目录名称为"镜像名称.tag"，如 ”ubuntu.16.04-ssh"
3. 如果镜像编译过程中需要使用一些大的文件，不要提交到本项目，而是提供一个下载这些文件的脚本
4. 同时提交readme文件
